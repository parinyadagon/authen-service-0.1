OAuth2 Authorization Code Flow Test

This page simulates a client application testing OAuth2 flow with your authentication service.

Instructions:

  1. Make sure your authentication service is running on http://localhost:8080
  2. Make sure your frontend is running on http://localhost:5173
  3. Fill in the form above and click "Start OAuth2 Flow"
  4. You'll be redirected to the authorization page
  5. After approval, you'll be redirected back with an authorization code

Expected Flow:

1. Client → Authorization Server: /oauth/authorize
2. User authenticates and grants permission
3. Authorization Server → Client: redirect with code
4. Client → Authorization Server: /oauth/token (exchange code for tokens)
